# Stanford_CodeInPlace_Projects

- [Karel Adventure](https://codeinplace.stanford.edu/cip6/share/gnLhoLan71GUvLmxd43t): A Mario-style game made with Karel concepts, where the character moves through a level with obstacles and goals.
- [Cube, Cube. Cube...](https://codeinplace.stanford.edu/cip6/share/6VUJOqW5tqAphABirfVy): An interactive cube-rendering project built using vertices and edges. When the user clicks the cube, it changes colors.
- [CLI Drawing with AI](https://codeinplace.stanford.edu/cip6/share/Ljrav5ciUc6wjjPCwAxM): A command-line drawing project that uses the Code in Place AI API to generate drawings from user input.
